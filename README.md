# 💼 Dhilip's Portfolio

A responsive, modern personal portfolio website showcasing projects, skills, and contact information. Built with HTML, CSS, and JavaScript to demonstrate web development expertise.

![Portfolio Website](wallpaperflare.com_wallpaper\(9\).jpg)

---

## 🎯 Overview

This is a single-page portfolio application designed to present professional achievements, technical skills, and completed projects. The portfolio features a clean, modern design with smooth animations and interactive elements that provide an engaging user experience.

**Live Demo:** [View Portfolio](https://dhilipmpms.github.io/portfolio/)

---

## ✨ Features

- **Responsive Design** - Fully responsive layout that works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations** - AOS (Animate On Scroll) library for smooth entrance animations
- **Interactive Navigation** - Mobile-friendly navigation with smooth scroll links
- **About Section** - Professional introduction with personal details
- **Skills Display** - Visual progress bars showing proficiency in various technologies
- **Featured Projects** - Showcase of completed projects with GitHub links and live demos
- **Contact Form** - Functional contact form with form validation
- **Social Links** - Integration with GitHub, Instagram, and other social platforms
- **Mobile Navigation** - Hamburger menu for mobile devices

---

## 🛠️ Technologies Used

### Frontend Stack
- **HTML5** - Semantic markup and structure
- **CSS3** - Styling with custom designs and animations
- **JavaScript** - Interactive features and form handling

### Libraries & Frameworks
- **Bootstrap 5** - Responsive grid and component framework

## 🚀 Quick Start

### Prerequisites
- A modern web browser
- Basic knowledge of HTML, CSS, and JavaScript

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/dhilipmpms/portfolio.git
   cd portfolio
   ```

2. **Open the portfolio**
   - Simply open `index.html` in your web browser
   - Or use a local server:
     ```bash
     python -m http.server 8000
     # Then visit http://localhost:8000
     ```

## 🎨 Customization

### To customize this portfolio:

1. **Edit Personal Information**
   - Update content in the About section (index.html, lines 95-102)
   - Modify contact details (index.html, lines 677-693)

2. **Change Skills**
   - Edit the Skills section (index.html, lines 200-257)
   - Update progress bars with your proficiency levels

3. **Add/Update Projects**
   - Modify the Featured Projects section (index.html, lines 491-575)
   - Add project images to `assets/img/`

4. **Styling**
   - Main styles are in `assets/css/style.css`
   - Customize colors, fonts, and layouts as needed
